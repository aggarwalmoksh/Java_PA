# Java Programming Abstraction

A collection of my Java solutions to programming and algorithmic problems, primarily from **LeetCode**, along with coursework completed during the Programming Abstraction course.

Each problem is stored in its own directory and includes:
- 📄 Problem statement (`README.md`)
- ☕ Java solution
- 🔄 Automatically synchronized after accepted submissions

## Repository Structure

```

│   └── Code
```
